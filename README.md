STUDENT COURSE REGISTRATION SYSTEM

This project is a simple console-based Student Course Registration System developed
as part of a Software Engineering (SEN) assignment. The system allows students to
register, log in, view available courses, and register a course.

FEATURES
- Student registration
- Student login
- Display available courses
- Course registration

TECHNOLOGY USED
- Python programming language

PROJECT FILES
- student_course_registration.py

HOW TO RUN THE PROJECT
1. Make sure Python is installed on your system
2. Open a terminal or command prompt
3. Navigate to the project folder
4. Run the command:
   python student_course_registration.py

PURPOSE OF THE PROJECT
This project demonstrates the application of the Software Development Life Cycle
(SDLC) using a simple and understandable software system.
